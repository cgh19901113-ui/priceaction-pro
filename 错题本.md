### 错误 #8: Vercel Routes 配置错误
**时间**: 2026-04-10 00:37  
**错误信息**:
```
404: NOT_FOUND
Code: DEPLOYMENT_NOT_FOUND
```

**原因**:
- routes 配置用了 `builds` 字段（旧版本）
- 应该用 `functions` 字段（新版本）
- `/api/(.*)` 路由没有正确匹配

**修复**:
```json
// ❌ 错误
{
  "version": 2,
  "builds": [
    {"src": "api/index.py", "use": "@vercel/python"}
  ],
  "routes": [
    {"src": "/api/(.*)", "dest": "api/index.py"}
  ]
}

// ✅ 正确
{
  "version": 2,
  "functions": {
    "api/index.py": {
      "runtime": "@vercel/python@3"
    }
  },
  "routes": [
    {"src": "/api", "dest": "api/index.py"},
    {"src": "/api/(.*)", "dest": "api/index.py"}
  ]
}
```

**教训**: Vercel Python 用 `functions` 不是 `builds`，且需要同时配置 `/api` 和 `/api/(.*)`
