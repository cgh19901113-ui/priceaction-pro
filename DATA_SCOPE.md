# 数据范围说明

## 为什么只做国内 A 股/港股？

### 1. 策略来源
本策略基于 **秋生 Trader(@Hoyooyoo)** 的价格行为理论，其分析对象主要是：
- A 股 (缅 A)
- 港股 (恒生科技等)
- 部分商品期货 (黄金/白银)

原博明确表示：
> "本人没有群、没有联系方式、不私信、更没有交易信号！"
> "认准唯一账号，遇到冒充我头像名字在我推文评论区发推的骗子号，请立刻举报！"

### 2. 数据可获取性

| 市场 | 数据源 | 完整性 | 成本 |
|------|--------|--------|------|
| A 股 | Tushare Pro/聚宽/akshare | ✅ 完整 | 免费/低价 |
| 港股 | Yahoo Finance/AKShare | ✅ 完整 | 免费 |
| 美股 | Yahoo Finance | ⚠️ 部分缺失 | 免费 |
| 币圈 | 多个 API | ✅ 完整 | 免费 |

**问题**：
- yfinance 的美股/加密货币数据不完整 (尤其是 2025 年)
- A 股数据用 akshare 可获取完整历史数据

### 3. 用户群体

**目标用户**:
- 国内散户投资者
- A 股/港股交易者
- 熟悉秋生 Trader 风格的粉丝

**不服务**:
- 美股交易者
- 加密货币投资者
- 期货/外汇交易者

### 4. 合规性

**国内数据**:
- 使用公开数据源 (akshare/Tushare)
- 不涉及跨境数据传输
- 符合国内数据合规要求

**国外数据**:
- 可能涉及跨境数据传输
- 需要额外的合规审查

## 回测策略

### 样本要求
- **最少 1000 只 A 股** 才有统计意义
- 覆盖不同行业/市值/风格
- 回测周期至少 1 年

### 执行方式
```bash
# 批量回测 1000 只 A 股
python strategy/backtest_1000.py
```

### 输出结果
- 通过率 (%): 策略给出"观察池"的比例
- 准确率 (%): 通过股票中赚钱的比例
- 平均收益：通过股票 3 日平均收益
- Top 10 推荐股票
