# 技术决策记录（ADR）

## ADR-001: Twitter 数据收集方案选择

**日期**: 2026-04-05  
**状态**: 已决策（等待执行）

### 背景

需要收集 10 位 Twitter 技术分析博主的历史推文（目标 10000+ 条），用于 PriceAction Pro 策略回测。

### 探索的方案

#### 1. snscrape 库
- **结果**: ❌ 不可行
- **原因**: Windows 上安装失败（依赖冲突）
- **时间**: 2026-04-05 22:39

#### 2. Nitter API
- **结果**: ❌ 不可行
- **原因**: 所有实例都无法访问（网络限制）
- **时间**: 2026-04-05 22:45

#### 3. Playwright 浏览器自动化
- **结果**: ✅ 可行（需一次手动登录）
- **优点**: 数据完整、可自动化、支持增量
- **缺点**: 首次需要 2 分钟手动登录
- **时间**: 2026-04-05 23:00

#### 4. Jina AI RSS
- **结果**: ⚠️ 数据不足
- **优点**: 无需登录、完全自动化
- **缺点**: 每位博主仅 14 条，共 2 条含股票
- **时间**: 2026-04-05 23:51

### 决策

**选择方案 3（Playwright + 一次手动登录）**

**理由**:
1. 数据量充足（10000+ 条 vs RSS 的 2 条）
2. 只需 2 分钟手动操作（一次性）
3. 之后永久自动化
4. 支持增量更新

### 执行计划

```bash
# 1. 登录（2 分钟手动）
python setup_login.py

# 2. 自动收集（50 分钟）
python run_auto.py

# 3. 自动导出 + 回测
python run_auto.py --export
python ..\..\strategy\backtest_twitter.py
```

### 备选方案

如果用户没有 Twitter 账号：
- 创建雪球网爬虫（中国版 Twitter）
- 创建微博财经博主爬虫
- 手动整理 Excel 样本

## ADR-002: 数据存储格式

**日期**: 2026-04-05  
**状态**: 已实施

### 决策

**原始数据**: JSONL 格式（每行一个 JSON 对象）
- 文件：`data/twitter/{username}.jsonl`
- 优点：支持增量追加、易去重、节省空间

**回测数据**: JSON 格式（数组）
- 文件：`strategy/twitter_recommendations.json`
- 优点：易读取、已去重、已排序

**进度跟踪**: JSON 格式
- 文件：`progress.json`
- 内容：每位博主的收集数量

### 数据结构

```json
// 原始推文
{
  "tweet_id": "1234567890",
  "timestamp": "2025-11-12",
  "username": "Hoyooyoo",
  "content": "中国中免 601888，大周期多头向上",
  "symbols": ["601888.ss"],
  "sentiment": "bullish"
}

// 回测推荐
{
  "symbol": "601888.ss",
  "date": "2025-11-12",
  "blogger": "@Hoyooyoo",
  "note": "大周期多头向上",
  "sentiment": "bullish",
  "tweet_id": "1234567890"
}
```

## ADR-003: 股票代码识别策略

**日期**: 2026-04-05  
**状态**: 已实施

### 决策

**多层次识别**:

1. **中文股票名映射**（优先级最高）
   - 维护映射表（贵州茅台→600519.ss）
   - 支持简称（茅台→600519.ss）

2. **正则表达式匹配**
   - A 股：`\d{6}` → 自动判断 .ss/.sz
   - 港股：`\d{5}.hk`
   - 美股：`\$[A-Z]{1,5}` 或纯字母代码

3. **去重**
   - 使用 set 去重
   - 一条推文可能提及多只股票

### 映射表（已实现）

```python
NAME_TO_SYMBOL = {
    '贵州茅台': '600519.ss', '茅台': '600519.ss',

    '比亚迪': '002594.sz',
    '中国中免': '601888.ss',
    '重庆啤酒': '600132.ss',
    '特斯拉': 'TSLA',
    '腾讯': '00700.hk',
```
